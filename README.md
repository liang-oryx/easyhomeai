Open source Home AI System. Control home light、switch、air conditioner etc...
开源家庭人工智能系统, 可以控制灯、开关、空调，需要使用配套的开源网关等智能硬件。 
详情请访问 www.easyhomeai.com 

1. /web web ui source code.(web界面的源码)。
2. /yodalite the hardware svr source code. (硬件服务器源代码)。
3. /lib (Apache HttpComponents and netty lib). 
4. /bin (可执行文件)
5. /face (人脸识别源码)
6. /voice (语音识别源码)
7. /doc 
8. /demo 文档
